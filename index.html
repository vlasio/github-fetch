github—fetch

A fastfetch-style profile generator. Pick a logo, edit the fields (or paste a CSV), and export. Use SVG for your GitHub README — it's the only format that keeps the colours.

github-fetch~/.config/github-fetch
logo
identity
colours
fields
csv

CSV columns are label,value. Wrap values containing commas in "double quotes". Importing replaces the field list.

github-fetchoctocat@github: ~
export

For your GitHub profile: download the SVG, commit it to your username/username repo, and embed it in README.md with <img src="github-fetch.svg">. Plain text / markdown drop into a code block but render monochrome (GitHub ignores ANSI colour).
